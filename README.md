# fourwheel

A Car on Rent web application named Fourwheel made with the use of django framework, python language, sqilite database, Javascript, html and css.


## Features

- Easy Navigation
- Car Search
- Car List
- Add car
- Booking car 
- Order Updation 
- Admin View
- Order History
- Registraion 
- login
